> 표준: ISO 8502-4(노점·결로 위험 평가), ASTM E337(습도), 도료 TDS의 환경조건 한계
> ⚠️ 구체적 한계치(최저 강재온도, 습도 상한)는 제품 TDS·시방 우선

## 1. 왜 측정하는가 (기초)

도장 품질의 절반은 **도장하는 순간의 환경**에서 결정된다. 핵심은 **강재 표면에 결로(이슬)가 생기지 않게** 하고, 도료가 정상 건조·경화할 수 있는 온도·습도 범위를 지키는 것이다.

- 강재온도가 **노점(Dew point)** 보다 낮거나 너무 가까우면 표면에 **수분이 응축** → 부착 불량·블리스터·플래시러스트·백화.
- 그래서 업계 표준 룰: **강재온도 ≥ 노점 + 3℃** (대부분의 시방·TDS 기준. 일부는 더 엄격).

## 2. 핵심 개념 (심화)

| 항목 | 정의 |
|---|---|
| **대기온도(Air temp.)** | 주변 공기 온도 |
| **상대습도(RH%)** | 현재 공기가 포화 대비 머금은 수증기 비율 |
| **노점(Dew point)** | 공기를 식힐 때 수증기가 응축되기 시작하는 온도 |
| **강재온도(Surface/Substrate temp.)** | 실제 도장면(강재)의 온도 — 복사·바람으로 대기온도와 다를 수 있음 |
| **ΔT (델타T)** | 강재온도 − 노점. 이 값이 결로 안전 여유 |

> 핵심: **대기온도가 아니라 '강재온도'와 노점의 차이**가 결로를 결정한다. 야간·새벽, 그늘진 강재, 두꺼운 블록은 대기보다 차가워 결로 위험이 크다.

## 3. 준비물

- 통합 노점계(Dewpoint meter, 예: 표면 온도 프로브 + 습도 센서 일체형) — 교정 유효 확인
- 또는: 건습구 습도계(Whirling/Assmann) + 표면 온도계 + 노점 환산표
- 자기온습도기록계(연속 기록, 탱크 도장 시)

## 4. 시험 절차 (상세 매뉴얼)

### 4.1 통합 노점계 사용 (현장 표준)
1. 장비를 측정 환경에 **충분히 순응(Acclimatize)** 시킨다(급격한 온도차에서 가져온 직후는 오차).
2. 대기온도·상대습도를 측정 → 장비가 **노점을 자동 산출**.
3. **표면 온도 프로브**를 강재면에 밀착(자석/접촉식)하여 **강재온도** 측정 — 도장할 바로 그 면에서.
4. **ΔT = 강재온도 − 노점** 확인. 통상 **≥ 3℃** 인지 판정.

### 4.2 건습구 습도계 사용 (대체)
1. 습구 거즈를 증류수로 적시고 규정대로 통풍/회전시켜 건구·습구 온도 측정.
2. 건구·습구 온도차로 **상대습도·노점을 환산표(또는 ISO 8502-4 표)** 에서 읽는다.
3. 별도 표면 온도계로 강재온도 측정 → ΔT 산출.

### 4.3 측정 위치·시점
- 도장할 구역의 **대표 위치 + 가장 차가운 면(그늘·하부·두꺼운 부재)**.
- **도장 직전 + 도장 중 주기적**으로(환경은 변한다). 탱크·대형 블록은 연속 기록계 권장.

## 5. 결과 검토 / 합부 판정

판정은 보통 **세 가지를 동시에** 본다:

1. **결로 안전**: 강재온도 ≥ 노점 + 3℃ (시방 기준)
2. **온도 범위**: 강재·대기 온도가 TDS의 최저~최고 적용온도 안 (저온이면 경화 정지, 고온이면 가사시간 급단축·드라이스프레이)
3. **습도 범위**: RH가 TDS 상한 이하 (특히 우레탄 등 수분 민감 도료, 무기징크처럼 **하한**이 있는 경우도 있음 — 제품 특성 확인)

- 어느 하나라도 벗어나면 **도장 보류**. 가열/제습/환기 등으로 조건을 만든 뒤 재측정.
- 기록: 대기온도, RH, 노점, 강재온도, ΔT, 시각, 위치, 장비, 판정. (탱크는 연속 그래프 첨부)

## 6. 자주 하는 실수

| 실수 | 결과 | 예방 |
|---|---|---|
| 대기온도만 보고 판정 | 결로 간과(강재가 더 차가움) | 반드시 강재온도로 ΔT |
| 장비 미순응 | 온도/습도 오차 | 환경 순응 후 측정 |
| 1회만 측정 | 변화 놓침 | 도장 중 주기적/연속 |
| 가장 차가운 면 누락 | 국부 결로 | 그늘·하부도 측정 |
| 무기징크 하한 습도 간과 | 경화 불량 | 제품별 습도 범위 확인 |

## 7. 검사원 체크리스트
- [ ] 장비 교정·환경 순응
- [ ] 대기온도·RH·노점·강재온도 측정, ΔT 산출
- [ ] ΔT ≥ 3℃(시방) 확인
- [ ] 온도·습도가 TDS 적용 범위 내인지(상·하한)
- [ ] 가장 차가운 면 포함 복수 위치, 도장 중 재측정/연속기록
- [ ] 전 항목 기록, 미충족 시 도장 보류·환경 개선

## 8. 핵심 요약
1. 결로를 결정하는 것은 **강재온도와 노점의 차(ΔT)** — 대기온도가 아니다.
2. 표준 룰: **강재온도 ≥ 노점 + 3℃**.
3. 온도(경화)·습도(수분 민감/무기징크 하한)·결로 — **세 조건 동시 판정**.
4. 가장 차가운 면을 포함해 복수점·주기적으로 측정.
5. 한계치는 TDS/시방 우선, 미충족 시 도장 보류.

## 9. 확인 문제
1. 대기온도가 충분히 높아도 결로가 생길 수 있는 이유는? (강재가 더 차가울 수 있음)
2. ΔT의 정의와 표준 안전 기준은? (강재온도−노점 ≥ 3℃)
3. 무기징크가 다른 도료와 습도 관리에서 다른 점은? (경화에 하한 습도 필요)
