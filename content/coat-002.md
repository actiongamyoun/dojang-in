## 1. 인산염 피막이란 (기초)

인산염 피막(Phosphate conversion coating)은 금속 표면을 **인산염 용액과 반응**시켜 표면에 **불용성 인산염 결정층**을 형성하는 화학 전처리다. 이 피막은:

- **도료 부착의 바탕**(앵커 역할 + 화학적 친화)을 만들고,
- **방청(부식 지연)** 을 보조한다.

자동차 차체, 가전, 금속 부품 도장 전처리의 표준 공정이며, 조선·산업에서는 워시프라이머(인산 함유)나 부품 전처리 형태로 만난다.

> '변환 피막(Conversion coating)'이라 부르는 이유: 금속 표면 자체를 **다른 화합물로 변환**시켜 피막을 만들기 때문(도료처럼 위에 얹는 게 아님).

## 2. 종류와 원리 (심화)

### 2.1 대표 종류
| 종류 | 특징 | 용도 |
|---|---|---|
| **아연계 인산염(Zinc phosphate)** | 치밀한 결정, 우수한 도장 바탕·방청 | 자동차·고급 도장 전처리 |
| **철계 인산염(Iron phosphate)** | 얇고 경제적, 결정성 낮음(비정질) | 일반 도장 전처리 |
| **망간계 인산염(Manganese phosphate)** | 두껍고 내마모·윤활성 | 기계부품(내마모, 길들임) |

### 2.2 형성 원리
1. 금속 표면이 인산 용액에 의해 **국부적으로 에칭**(미세 용해)되며 표면 pH가 상승.
2. 용액 중 금속 인산염이 **과포화 → 표면에 불용성 인산염 결정으로 석출**.
3. 표면에 균일한 인산염 결정층 형성 → 다공·결정 구조가 도료를 물게 함.

- 통상 공정: 탈지 → 수세 → (표면조정) → 인산염 처리 → 수세 → (후처리/봉공) → 건조.
- **표면조정(Surface conditioning)**: 인산염 처리 전 결정 핵을 미세화해 치밀한 피막을 얻는 단계(아연계에서 중요).

## 3. 검사 항목 (실무)

인산염 피막은 도장 검사원이 직접 형성하지는 않지만, **전처리 품질을 확인**하는 관점에서 본다:

- **피막 중량/두께(Coating weight, g/m²)**: 사양 범위 충족 여부(과소→부착·방청 부족, 과대→피막 취성·부착 저하).
- **결정 균일성·외관**: 균일한 회색 결정. 얼룩·미형성부·과대결정 확인.
- **수세·건조**: 잔류 산/염 제거(잔류물은 후속 도막 결함 원인), 완전 건조.
- **후속 도장 적합성**: 피막 위 도장의 부착(크로스컷 등).

## 4. 인산염 피막 vs Passivation (혼동 주의)

| 구분 | 인산염 피막 | Passivation(부동태화) |
|---|---|---|
| 대상 | 주로 탄소강·아연·알루미늄 | 주로 스테인리스강 |
| 원리 | 인산염 결정층을 **형성(추가)** | 표면 철분 제거로 **부동태 산화막 강화** |
| 결과물 | 눈에 보이는 결정 피막 | 보이지 않는 크롬산화막 |
| 주 목적 | 도장 바탕 + 방청 | 내식성 회복(도장 전제 아님) |

> 자세한 Passivation은 별도 문서 참조. 둘 다 '표면 화학 처리'지만 **원리와 대상이 다르다**.

## 5. 대표 결함과 원인

| 결함 | 원인 | 대책 |
|---|---|---|
| 피막 미형성/얼룩 | 탈지 불량, 용액 농도/온도 이탈 | 공정 파라미터 관리 |
| 과대 결정·취성 | 표면조정 미흡, 과처리 | 표면조정·시간 관리 |
| 후속 도막 부착 불량 | 잔류 염/산, 과대 피막 | 충분 수세, 중량 관리 |
| 발청(백화/적청) | 수세·건조 불량, 잔류 수분 | 완전 수세·건조 |

## 6. 검사원 체크리스트
- [ ] 공정 종류 확인(아연/철/망간계)와 사양 정합
- [ ] 피막 중량/외관/결정 균일성
- [ ] 탈지·수세·건조 품질(잔류물·수분)
- [ ] 후속 도장 부착(크로스컷 등)
- [ ] Passivation과 혼동 없이 적용 대상·목적 확인

## 7. 핵심 요약
1. 인산염 피막 = 금속 표면을 **불용성 인산염 결정층으로 변환**하는 전처리(부착 바탕+방청).
2. 종류: 아연계(고급 도장)·철계(경제형)·망간계(내마모).
3. 핵심 공정 변수는 탈지·표면조정·농도/온도·수세/건조.
4. 검사 포인트: 피막 중량·균일성·잔류물·후속 부착.
5. **Passivation과 다르다** — 인산염은 '피막 형성', Passivation은 '부동태막 강화'.
