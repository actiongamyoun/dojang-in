## 1. 왜 부식을 알아야 하나 (기초)

도장의 제1 목적이 **부식 방지**다. 부식이 왜·어떻게 일어나는지 알아야 **도장 시스템이 무엇을 막는지, 어디서 실패하는지**를 이해한다. 검사 중 발견하는 발청·박리·블리스터의 **원인을 부식 메커니즘으로 설명**할 수 있어야 제대로 된 진단이다.

## 2. 부식의 전기화학 (심화)

철의 부식은 **전기화학 반응**이다. 네 요소가 모두 있어야 진행되고, **하나만 제거해도 멈춘다**:

| 요소 | 역할 |
|---|---|
| **양극(Anode)** | 금속이 이온으로 용출(산화): Fe → Fe²⁺ + 2e⁻ |
| **음극(Cathode)** | 환원 반응(중성: O₂+2H₂O+4e⁻→4OH⁻) |
| **전해질(Electrolyte)** | 이온 이동 매질(물+염) |
| **전자 통로(Metallic path)** | 양극-음극 전자 이동(금속 자체) |

> 도장의 본질: **전해질·산소 차단(Barrier)**, 또는 **양극을 대신 희생(아연)**, 또는 **부동태화(억제)**. 부식 4요소 중 하나 이상을 끊는 것.

## 3. 부식의 형태(Forms) — 검사 시 식별 (실무 핵심)

| 형태 | 특징 | 검사 시사점 |
|---|---|---|
| **균일부식(Uniform)** | 전면에 고르게 | 일반 대기부식, 도막 차단으로 대응 |
| **공식(Pitting)** | 국부 깊은 구멍 | 염화물(SUS·알루미늄) 위험, 깊어 위험 |
| **틈부식(Crevice)** | 틈·겹침부 | 볼트·겹침·퇴적물 아래, 설계/도장 취약 |
| **갈바닉(Galvanic)** | 이종금속 접촉 | 비금속 쪽 가속 부식(절연·도장 필요) |
| **입계부식(IGC)** | 입계 따라(예민화 SUS) | 용접부, L그레이드 대응 |
| **응력부식균열(SCC)** | 응력+환경(염화물 등) | SUS 보온부 등, 균열성 |
| **침식부식(Erosion)** | 고유속·캐비테이션 | 펌프·해수배관·프로펠러 |
| **미생물부식(MIC)** | 미생물 관여 | 정체수·탱크 |
| **도막하 부식(Underfilm)** | 도막 아래 진행 | 염분·손상부서 시작, 언더커팅 |

## 4. 도장 관련 부식 현상 (실무)

### 4.1 도막하 부식·언더커팅(Undercutting)
- 도막 손상부·핀홀에서 시작해 **도막 아래로 부식이 번짐**. 염분 잔류가 가속. 검사 시 박리부 주변 적청·들뜸으로 식별.

### 4.2 블리스터링(삼투)
- 도막하 수용성염이 **삼투압으로 수분을 끌어들여** 도막을 부풀림. → 염분 관리(Bresle)의 이유.

### 4.3 갈바닉 — 이종금속 경계
- 강재+스테인리스/알루미늄/구리 접촉부는 **卑금속이 가속 부식**. 절연·도장·설계로 대응. 선박 관통부·이종 배관 경계 주의.

### 4.4 음극방식과의 관계
- 아연 프라이머(희생방식), 선체 ICCP/아노드(음극방식)는 **양극 반응을 대신 떠안아** 강재를 보호. 도장+음극방식의 협업.

## 5. 환경 부식성 분류 (심화)

- **ISO 9223 / ISO 12944-2**: 대기 부식성을 **C1(실내 건조)~C5/CX(가혹 해양·산업)**, 침적 **Im1~Im4** 로 분류.
- 환경이 가혹할수록 더 강한 도장 시스템·두께·수명 등급 요구(→ 표준 레퍼런스/선박구조 문서 연계).
- **염분·습도·온도·오염**이 부식속도를 좌우.

## 6. 검사 시 부식 식별·기록 (실무)

- 발청 형태(균일/공식/도막하) 식별 → 원인 추정(염분/손상/이종금속/환경).
- **ISO 4628** 시리즈로 녹·블리스터·크랙 등급 정량(열화 평가).
- 보수 범위 판단: 표면 발청 vs 도막하 진행(언더커팅)은 처리 깊이가 다름.
- 기록: 형태·범위·등급·추정원인·사진.

## 7. 대표 문제와 원인

| 현상 | 부식 원인 | 도장 대응 |
|---|---|---|
| 도막하 적청·언더커팅 | 손상부·염분서 도막하 부식 | 염분관리·손상부 보수·차단 |
| 블리스터 | 삼투(수용성염) | Bresle 관리·표면처리 |
| 이종금속부 가속부식 | 갈바닉 | 절연·도장·설계 |
| SUS 용접부 부식 | 예민화·heat tint | L그레이드·피클/부동태 |
| 해수배관 공식 | 염화물·유속 | 내식재·라이닝·음극방식 |

## 8. 검사원 체크리스트
- [ ] 부식 4요소 관점에서 도장 시스템 역할 이해
- [ ] 발청 형태 식별(균일/공식/틈/갈바닉/도막하)
- [ ] 도막하 부식·언더커팅·블리스터 구분
- [ ] 이종금속 경계·관통부 갈바닉 주의
- [ ] ISO 4628로 열화 등급화, 추정원인 기록
- [ ] 환경 부식성(C/Im)과 사양 정합 확인

## 9. 핵심 요약
1. 부식은 **전기화학(양극·음극·전해질·전자통로)** — 도장은 이 중 하나 이상을 끊는다(차단/희생/억제).
2. 형태별 식별이 진단의 출발: 균일·**공식**·틈·**갈바닉**·입계·SCC·침식·MIC·**도막하**.
3. 도장 실패와 직결: **도막하 부식/언더커팅, 블리스터(삼투), 갈바닉(이종금속)**.
4. **염분·손상부**가 도막하 부식·블리스터의 흔한 시발점 → Bresle·보수 관리.
5. 환경 부식성(**C1~CX/Im**)이 시스템·두께·수명을 좌우. 열화는 **ISO 4628**로 등급화.
