> 관련: ISO 8502-2(염화물), SSPC-Guide 15, US Navy NSI 등 / ⚠️ 허용 기준치는 시방 우선

## 1. 왜 측정하는가 (기초)

강재 표면에 남은 **수용성 염(특히 염화물 Cl⁻, 황산염)** 은 도막 아래에서 **삼투압에 의한 수분 유입 → 블리스터링(부풀음)·도막하 부식**을 일으키는 가장 흔한 조기 도장 실패 원인이다. 눈에 보이지 않기 때문에 **측정으로만 관리**할 수 있다. 해안 야적·해상 운송이 많은 조선 환경에서는 특히 중요하다.

- 측정 시점: **블라스트 직후~도장 직전**(취약부 포함). 도장 후에는 의미가 없다.
- 측정 대상: 도장 전 강재면(주로). 보수 도장 시 기존 도막 위 염분도 관리 대상이 될 수 있음.

## 2. 측정 원리 (심화)

![Bresle 측정 절차](img/01_bresle_salt.svg)

Bresle법은 표면의 가용성 염을 **물에 녹여 회수한 뒤, 그 물의 전기전도도로 염의 양을 추정**하는 방법이다.

1. 표면에 **Bresle 패치(셀룰로오스 라텍스 패치)** 를 밀착 부착 → 패치와 표면 사이에 밀폐된 시험 셀 형성.
2. 패치 안에 **탈이온수(증류수)** 를 주입하고 주사기로 **반복 순환**시켜 표면의 염을 물에 녹인다.
3. 일정 시간/횟수 후 물을 회수 → **전도도계(Conductivity meter)** 로 전기전도도(μS/cm) 측정.
4. 측정값을 면적으로 환산하여 **단위면적당 염화물(또는 가용성 염) 표면밀도(mg/m²)** 로 표현.

> 핵심: 전도도는 '모든 이온의 총량'을 본다. 염화물만 따로 알아야 하면 별도의 염화물 전용 측정(ISO 8502-2 등)을 병행한다.

## 3. 준비물

- Bresle 패치(시험면적이 규정된 표준 패치)
- 탈이온수/증류수(전도도가 충분히 낮은 것), 주사기(니들)
- 전도도계(온도보정 기능 권장) — 교정 유효 확인
- 비커/시료 용기, 장갑, 기록지

## 4. 시험 절차 (상세 매뉴얼)

### Step 0 — 사전 준비
- 전도도계 교정 상태 확인. 사용할 **탈이온수의 바탕 전도도(블랭크)를 먼저 측정·기록**한다(나중에 차감).
- 측정 위치 선정: 대표면 + **취약부(용접부, 모서리, 구석, 핸드그라인딩부)**. 1점이 아닌 복수점.

### Step 1 — 패치 부착
- 패치 뒷면 보호지를 떼고 표면에 **기포 없이 밀착**. 패치 모서리가 들뜨면 누수로 값이 왜곡된다.

### Step 2 — 물 주입 및 순환
- 주사기로 규정량의 탈이온수를 패치 내부로 주입(패치 내 공기는 빼고 물을 채움).
- 규정된 시간 동안 주사기로 **물을 넣고 빼기를 반복**(통상 여러 회)하여 표면 염을 충분히 용해. 이때 표면을 긁지 않도록 주의.

### Step 3 — 시료 회수 및 측정
- 순환이 끝난 물을 주사기로 모두 회수 → 전도도계로 측정. (셀 내 잔류 없이 회수)
- 온도가 전도도에 영향을 주므로 온도보정 기능을 쓰거나 수온을 기록.

### Step 4 — 계산 및 환산
- **보정 전도도 = 측정 전도도 − 블랭크(물 자체) 전도도**
- 보정 전도도를 **패치 시험면적과 사용한 물의 양**을 반영해 표면밀도로 환산.
  - 전도도(μS/cm)를 염화물 표면밀도(mg/m²)로 바꾸는 변환은 **표준/장비 매뉴얼의 환산식 또는 환산표**를 사용한다(임의 계산식 금지).
- 복수점 결과를 기록하고 기준치와 비교.

## 5. 결과 검토 / 합부 판정

- **허용 기준은 시방에 따른다.** 일반적으로 탱크·발라스트 등 침적·중요부가 일반 외판보다 엄격하다.
  - 예: IMO PSPC(발라스트 탱크 보호도장 성능기준)는 수용성염(염화물 환산)의 상한을 규정하며, 이를 충족해야 한다 — **정확한 수치는 해당 PSPC/선주 시방 원문 확인**.
- 기준 초과 시: **담수 세척(Fresh water washing) 후 재측정**. 통과할 때까지 반복. 세척 후 재발청(Flash rust) 관리도 함께.
- 기록: 위치, 측정값(보정 후), 환산 표면밀도, 블랭크값, 수온, 장비/패치 정보, 판정.

## 6. 자주 하는 실수 (실무)

| 실수 | 결과 | 예방 |
|---|---|---|
| 블랭크 미차감 | 값 과대 → 불필요한 재작업 | 물 바탕 전도도 먼저 측정 |
| 패치 들뜸/누수 | 값 왜곡(과소/과대) | 기포 없이 밀착, 모서리 확인 |
| 오염된 탈이온수 사용 | 바탕 전도도 높아 부정확 | 저전도도 신선한 물 사용 |
| 단일점만 측정 | 취약부 누락 | 대표면+취약부 복수점 |
| 환산식 임의 적용 | 잘못된 mg/m² | 표준/장비 환산표 사용 |

## 7. 검사원 체크리스트
- [ ] 전도도계 교정·온도보정, 탈이온수 블랭크 측정
- [ ] 측정 시점(블라스트 직후~도장 직전), 복수 위치(취약부 포함)
- [ ] 패치 밀착·규정 순환시간 준수
- [ ] 보정 전도도→표면밀도 환산(표준 환산표 사용)
- [ ] 시방 기준치와 비교·판정, 초과 시 담수세척·재측정
- [ ] 전 항목 기록(값·위치·조건·장비)

## 8. 핵심 요약
1. 수용성염은 도막하 부식·블리스터의 주범 → **도장 직전 측정**으로만 관리 가능.
2. Bresle법 = 패치로 염을 물에 녹여 **전도도로 양을 추정**(ISO 8502-6/-9).
3. **블랭크 차감**과 **표준 환산표 사용**이 정확도의 핵심.
4. 기준치는 시방 우선(탱크·침적부가 더 엄격, 예: PSPC).
5. 초과 시 담수세척 후 재측정 — 통과까지 반복.
