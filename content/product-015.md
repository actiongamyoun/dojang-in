## 1. 왜 혼합이 중요한가 (기초)

중방식 도료의 대부분은 **2액형(주제+경화제)** 이다. 이들은 **혼합하는 순간 화학반응(가교)이 시작**되며, **혼합 품질이 도막 성능을 영구히 좌우**한다. 혼합비가 틀리거나, 덜 섞이거나, 가사시간을 넘긴 도료를 쓰면 **경화 불량·물성 미달**이 되고 이는 **되돌릴 수 없다**.

> 핵심: 혼합은 "대충 섞는" 작업이 아니라 **정량·정밀·시간 관리**가 필요한 품질 공정. 검사원의 혼합 입회가 탱크 도장의 기본인 이유.

## 2. 혼합비(Mixing Ratio) — 화학량론 (심화)

- 2액 도료의 혼합비(예: 주제:경화제 = 4:1)는 **화학량론(Stoichiometry)으로 설계**된 값이다 — 에폭시기와 경화제 활성수소가 정확히 반응하도록.
- **임의 변경 금지**: 경화제를 적게/많이 넣으면 미반응 성분이 남아 **내약품성·강도·부착이 영구 저하**.
- **세트 전량 혼합 원칙**: 캔을 부분 혼합하면 비율 오차 위험 → 가능하면 **세트(주제캔+경화제캔) 전량**을 섞는다. 부분 혼합 시 정확한 계량 필수.
- **부피비 vs 무게비**: TDS가 지정하는 단위(부피/무게)로 정확히. 혼동 금지.

## 3. 혼합 절차 (실무)

1. **개별 교반**: 주제를 먼저 단독으로 잘 저어 **침강된 안료를 균일화**(특히 징크는 침강 심함).
2. **계량·합류**: 정확한 비율로 경화제를 주제에 투입.
3. **기계 교반**: 저속 기계 교반기로 **균일해질 때까지**(색·점도 균일). 수동 부족분이 미경화 결함.
4. **용기 벽·바닥 긁기**: 덜 섞인 부분 없게.
5. **숙성(Induction)**: TDS가 요구하면 일정 시간 대기(사전 반응) 후 도장 — 특히 저온/일부 제품.
6. **희석**: 필요 시 지정 신너·지정 범위 내에서.

> **공기 혼입 주의**: 과격한 교반은 기포를 넣어 핀홀·포핑 유발. 적정 속도.

## 4. 가사시간(Pot Life) — 시간과의 싸움 (실무 핵심)

**가사시간**은 혼합 후 도료를 정상 사용할 수 있는 시간이다.

- **온도에 매우 민감**: **온도가 높을수록 급격히 짧아진다**(경험칙: 10℃ 상승 시 절반 수준 — 제품별 TDS 기준). 한여름 탱크 내부는 가사시간이 크게 단축.
- **점도로 판단 금지**: 가사시간이 지나면 **점도는 멀쩡해 보여도 가교가 진행**돼 부착·성능이 떨어진다. → **혼합 시각을 기록**하고 시간으로 관리.
- **발열(Exotherm)**: 혼합량이 많고 용기에 모여 있으면 반응열로 가사시간이 더 단축(통에서 굳음). 펌프로 빨리 소진.
- **가사시간 경과분은 폐기** — 추가 신너로 되살리려 하면 안 됨(물성 저하).

## 5. 희석(Thinning) (실무)

- **지정 신너·지정 희석률**(통상 일정 % 이내)만 사용. 임의 신너는 **백화·겔화·부착불량**.
- 희석 목적: 무화 개선·작업성(고온/에어리스 조건). 과희석은 **새깅·박막·용제 잔류**.
- 희석은 DFT에 영향(고형분 희석) → 두께 계산 반영.

## 6. 검사원 혼합 입회 포인트 (실무)

- **자재 확인**: 도료명·배치번호·유효기간(Shelf life), 보관온도.
- **혼합비**: TDS 비율·단위(부피/무게)·세트 전량/정확 계량.
- **교반**: 개별 교반(징크 침강)·기계 교반·벽바닥·균일성.
- **숙성**: 요구 시 시간 준수.
- **가사시간**: **혼합 시각 기록**, 온도 고려, 경과분 폐기.
- **희석**: 지정 신너·희석률.

## 7. 대표 문제와 원인

| 문제 | 원인 | 대책 |
|---|---|---|
| 미경화(연질·끈적) | 혼합비 오류, 교반 부족 | 정확 비율·기계 교반 |
| 부분 미경화 얼룩 | 벽/바닥 덜 섞임 | 긁어 균일 혼합 |
| 핀홀/포핑 | 과격 교반 기포 | 적정 교반·숙성 |
| 가사시간 후 도장 | 시간 관리 실패 | 혼합시각 기록·폐기 |
| 백화·겔화 | 비지정 신너 | 지정 신너만 |
| 새깅·박막 | 과희석 | 희석률 준수 |

## 8. 검사원 체크리스트
- [ ] 도료명·배치·유효기간·보관 확인
- [ ] 혼합비(TDS 단위)·세트 전량/정확 계량
- [ ] 개별 교반(징크 침강)→기계 교반→벽바닥 균일
- [ ] 숙성시간(요구 시) 준수
- [ ] 혼합 시각 기록·온도 고려·가사시간 경과분 폐기
- [ ] 지정 신너·희석률(DFT 영향 반영)

## 9. 핵심 요약
1. 2액형은 **혼합 순간 반응 시작** — 혼합 품질이 성능을 **영구히** 결정.
2. **혼합비는 화학량론** — 임의 변경 금지, **세트 전량/정확 계량**.
3. 절차: **개별 교반(침강)→정량 합류→기계 교반→벽바닥→(숙성)**. 과격 교반은 기포.
4. **가사시간은 온도에 민감**(높으면 급단축), **점도로 판단 금지** → 혼합 시각 기록·경과분 폐기.
5. **지정 신너·희석률**만 — 임의 신너는 백화/겔화, 과희석은 새깅/박막.
