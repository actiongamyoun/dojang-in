## 1. 핫딥 알루미나이징이란 (기초)

핫딥 알루미나이징(Hot-Dip Aluminizing, HDA)은 강재를 **용융 알루미늄(또는 알루미늄-실리콘 합금) 조에 침지**하여 표면에 알루미늄 피막을 입히는 공정이다. 용융아연도금(Galvanizing)의 '알루미늄 버전'이라고 보면 이해가 빠르다.

핵심 목적은 **내열(내산화) + 내식**:
- 알루미늄 피막이 고온에서 **치밀한 알루미나(Al₂O₃) 산화막**을 만들어 강재의 산화·스케일을 막는다.
- 배기계(머플러), 열교환기, 가열로 부품, 오븐 등 **고온 환경 강재**에 널리 쓰인다.

> 강재 코어의 강도 + 알루미늄의 내열·내식을 결합한 복합재. 아연도금이 '희생방식(내식) 중심'이라면, 핫딥 알루미늄은 '내열+차단 내식 중심'이라는 성격 차이가 있다.

## 2. Type 1 / Type 2 — 두 등급 (심화)

핫딥 알루미나이징 강판은 **코팅 조성에 따라 Type 1과 Type 2** 로 나뉜다(ASTM A463 등).

| 구분 | Type 1 | Type 2 |
|---|---|---|
| 코팅 조성 | **알루미늄 + 약 5~11% 실리콘 합금** | **순수(상용순도) 알루미늄** |
| 실리콘 역할 | 가열 시 **Fe-Al 합금층 성장 억제 → 내열성↑**, 피막 부착·균일성 향상 | (실리콘 없음) |
| 강점 | **내열성**(고온 산화 저항) | **내식성**(대기 부식 저항) |
| 성형성 | 상대적으로 우수 | 상대적으로 낮음 |
| 대표 용도 | 배기/머플러, 오븐, 가열로, 베이킹 팬 | 대기 내식 요구(루핑, 곡물 빈 등) |
| 사용 온도(개략) | 통상 약 600~677℃급까지(자료별 상이) | 내열은 Type 1보다 낮음 |

> 즉 **내열이 우선이면 Type 1(Al-Si), 내식이 우선이면 Type 2(순 Al)**. 실리콘 유무가 두 타입의 성격을 가른다.

### 피막 단면 구조
표면부터: **알루미늄(산화막)층 → 철-알루미늄(±실리콘) 금속간화합물(합금)층 → 강재 코어**.
- 이 **합금층(Intermetallic)** 이 야금학적 결합을 만들지만, 취성이 있어 과도한 변형 시 균열·박리 경향(성형성 한계의 원인).

### 확산법(Calorizing)과의 구분
- **확산 알루미나이징(Calorizing/Diffusion)**: 알루미늄 분말 속에서 가열해 알루미늄을 표면에 **확산 침투**시켜 합금층 형성. 핫딥보다 **금속간 결합이 더 강하고 고온에 유리**한 경우가 있으나 공정이 다르다.
- 핫딥은 '침지', 확산은 '분말 속 가열 확산' — **공정·피막 구조가 다르므로 검사 전 어느 공정인지 확인**.

## 3. 핫딥 알루미늄 위 도장이 까다로운 이유 (실무 핵심)

아연도금 위 도장과 유사한 원리적 어려움이 있다:

### 3.1 비활성·매끈한 알루미늄 표면
- 알루미늄 표면은 **안정한 산화막(Al₂O₃)** 으로 덮여 비활성 → 일반 도료가 잘 물지 않는다.
- 매끈한 표면은 기계적 투묘가 약하다.

### 3.2 표면처리(전처리)가 부착의 열쇠
| 방법 | 내용 |
|---|---|
| **스위프 블라스팅** | 비철연마재로 가볍게 쓸어 산화막 제거 + 조도 부여(과도 제거 금지) |
| **세정·탈지** | 오일·오염 제거 |
| **알루미늄 전용 전처리/에치 프라이머** | 알루미늄용 워시프라이머·전처리로 바탕 형성(크롬프리 제품 등) |

- 알루미늄은 양쪽성 금속(산·알칼리에 모두 반응)이라 **강알칼리 세정제 주의**(표면 침식).

### 3.3 도료 선정
- 알루미늄면 적합 프라이머(예: 에폭시 계열, 알루미늄용 전처리 프라이머). TDS의 '알루미늄/알루미늄도금면 적용' 항목 확인.
- 고온 부위면 **내열 도료(실리콘계 등)** 와의 적합성도 함께 고려.

## 4. 시공·검사 실무

### 4.1 도장 전
- **공정·타입 확인**: 핫딥(Type 1/2)인지 확산법인지. 고온 용도면 Type/내열 적합성 확인.
- 피막 상태: 손상·스패터·산화막 상태, (사양 시) 피막 두께.
- 전처리: 스위프 블라스트 조도/청정, 탈지, 전용 전처리 적용 확인.

### 4.2 도장 중·후
- 환경조건(노점), WFT/DFT, 재도장 간격.
- **부착 시험**(크로스컷/풀오프)으로 박리 리스크 검증 — 비활성면 도장이라 중요.

## 5. 대표 결함과 원인

| 결함 | 원인 | 대책 |
|---|---|---|
| 도막 박리 | 산화막·비활성면 직접 도장, 매끈함 | 스위프 블라스트·전처리 |
| 부착 불량 | 탈지 불량, 부적합 프라이머 | 탈지·알루미늄용 도료 |
| 고온부 박리 | 내열 부적합 톱코트 | 내열 사양 적용 |
| 표면 침식 | 강알칼리 세정 | 적합 세정제 |

## 6. 검사원 체크리스트
- [ ] 공정·타입 확정(핫딥 Type 1/2 vs 확산법), 고온 용도 적합성
- [ ] 피막 상태·두께(사양 시), 손상부
- [ ] 전처리: 스위프 블라스트 조도/청정, 탈지, 전용 전처리
- [ ] 알루미늄면 적합 프라이머·내열 적합성

## 7. 핵심 요약
1. 핫딥 알루미나이징 = 용융 알루미늄(±Si) 침지, **내열+내식** 복합재(아연도금의 알루미늄판).
2. **Type 1(Al-Si, 내열)** vs **Type 2(순 Al, 내식)** — 실리콘 유무가 성격을 가른다.
3. 피막 구조: 알루미나층→Fe-Al(±Si) 합금층→강재. 확산법(Calorizing)과는 공정이 다름.
4. 도장 난점은 아연도금과 유사: **비활성 산화막·매끈한 표면** → 스위프 블라스트+전처리+알루미늄용 도료.
5. 비활성면이라 **부착 시험으로 검증**하는 습관. 고온부는 내열 도료 적합성 확인.
