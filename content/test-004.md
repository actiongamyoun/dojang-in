> 표준: ISO 2409(Cross-cut test) / ASTM D3359 Method B(Cross-hatch) / ISO 16276-2
> 성격: 부착력을 **정량 측정이 아니라 '추정/등급화'** 하는 시험
> ⚠️ 합격 등급은 시방 우선

---

## 1. 무엇을 보는 시험인가 (기초)

크로스컷 시험은 도막에 **격자(바둑판) 모양으로 칼집**을 낸 뒤 **점착테이프로 떼어** 격자 사각형이 얼마나 떨어지는지를 표준 도해와 비교해 **등급(0~5 또는 0B~5B)** 으로 판정하는 시험이다.

- 장점: 빠르고 간편, 현장·생산 적합. 박막 도막의 층간/소지 부착을 추정.
- 한계: **부착 '강도(MPa)'를 측정하지 못한다.** 정량 강도가 필요하면 풀오프(돌리) 시험으로.
- 적용: 통상 **얇은 도막**에 적합. 두꺼운 도막에는 X컷(Method A)을 쓴다(후술).

## 2. 표준별 차이 — 가장 중요한 함정 (심화)

![크로스컷 등급](img/03_crosscut.svg)

### 2.1 ISO 2409 vs ASTM D3359 — 등급 번호가 '반대'다
- **ISO 2409**: 0(최상) → 5(최악). 'Gt 0 ~ Gt 5'로 표기.
- **ASTM D3359 Method B**: 5B(최상) → 0B(최악).
- 대응: **ISO 0 = ASTM 5B**(완벽), **ISO 5 = ASTM 0B**(최악). 번호 방향이 반대이므로 **반드시 어느 표준인지 명기**해야 혼선이 없다.

### 2.2 박리 면적 기준(공통 개념)
- 0(=5B): 박리 0%, 절단 가장자리 매끈
- 1(=4B): 교차점에서 미세 박리, ≤5%
- 2(=3B): 모서리/교차점 박리, 5~15%
- 3(=2B): 리본형 박리, 15~35%
- 4(=1B): 35~65%
- 5(=0B): 65% 초과(또는 그 이상)

### 2.3 칼날 간격(도막 두께에 따라)
- 얇을수록 촘촘하게: 대략 **≤60㎛ → 1mm 간격**, **61~120㎛ → 2mm 간격**.
- **약 121~250㎛ 이상의 두꺼운 도막에는 격자컷이 부적절** → ASTM에서는 X컷(Method A) 권장. 즉 **두께가 방법을 결정**한다.

## 3. 준비물

- 크로스컷 커터(다중 칼날 6blade) 또는 단일 예리 칼날 + 간격 가이드
- 규정 점착테이프(표준이 지정하는 점착력의 테이프)
- 부드러운 솔, 확대경(필요 시), 표준 등급 도해, 보수용 터치업 재료

## 4. 시험 절차 (상세 매뉴얼)

1. **위치 선정**: 평탄하고 대표적인 면(또는 의심부). 곡면·거친 면은 부적합 → X컷 고려.
2. **1차 절단**: 일정 압력으로 **6회 평행 절단**. 칼날이 **소지까지 닿되 과도하게 파지 않도록**(일정 속도, 일정 깊이).
3. **2차 절단**: 시편을 **90° 회전**시켜 다시 6회 절단 → 격자(예: 6×6) 형성.
4. **청소**: 부드러운 솔로 격자부의 떨어진 조각·먼지를 가볍게 제거.
5. **테이프 부착**: 규정 테이프를 격자 위에 기포 없이 부착하고 손가락/도구로 **균일하게 압착**(접촉 확보).
6. **테이프 제거**: 부착 후 규정 시간 내에 **약 60° 각도로 한 번에 신속히** 떼어낸다.
7. **판정**: 격자부 박리 면적·형태를 표준 도해와 비교하여 등급 결정(필요 시 확대경).

## 5. 결과 검토 / 합부 판정

- **합격 등급은 시방 우선.** 중방식 분야에서는 통상 **ISO Gt0~Gt1(= ASTM 4B~5B)** 을 합격으로 보는 경우가 많다.
- **파괴 위치 관찰**(원인 진단의 핵심):
  - 격자 사각형이 **층간(코트 사이)** 에서 떨어지면 → 재도장 간격 초과·아민 블러시·층간 오염 의심.
  - **소지(강재)에서** 떨어지면 → 표면처리 불량·염분·프로파일 부족 의심.
- **두꺼운 도막에서 격자컷이 깨끗이 안 되거나** 칼집이 소지까지 안 닿으면 결과 신뢰성 낮음 → X컷으로 전환.
- 기록: 표준/방법, 칼날 간격, 등급, 파괴 위치, 위치·도막·환경 정보, 사진. **시험부 보수(터치업)** 후 기록.

## 6. 자주 하는 실수

| 실수 | 결과 | 예방 |
|---|---|---|
| ISO/ASTM 혼용 표기 | 등급 정반대 해석 | 표준 명기 필수 |
| 두꺼운 도막에 격자컷 | 결과 부정확 | X컷으로 전환 |
| 칼집이 소지까지 미도달 | 박리 과소평가 | 일정 압력·깊이 확인 |
| 테이프 압착 불량 | 박리 과소(거짓 합격) | 균일 압착 |
| 제거 각도/속도 제각각 | 재현성 저하 | 약 60°·신속·일관 |

## 7. 검사원 체크리스트
- [ ] 도막 두께에 맞는 방법(격자 vs X컷)·칼날 간격
- [ ] 규정 테이프·균일 압착·약 60° 신속 제거
- [ ] 표준(ISO/ASTM) 명기하여 등급 판정
- [ ] 파괴 위치(층간/소지) 관찰·기록 → 원인 추정
- [ ] 시방 합격 등급과 비교, 시험부 보수
- [ ] 사진·조건 기록

## 8. 핵심 요약
1. 크로스컷은 부착을 **'등급으로 추정'**(강도 측정 아님). 정량은 풀오프로.
2. **ISO 0 = ASTM 5B**(최상), 번호 방향 반대 → **표준 명기 필수**.
3. **두께가 방법을 결정**: 두꺼우면(≈>125㎛) X컷.
4. 칼집은 소지까지, 테이프는 균일 압착·약 60° 신속 제거.
5. 파괴 위치(층간/소지)가 원인 진단의 단서. 합격 등급은 시방 우선.
