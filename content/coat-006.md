## 1. 스프레이 건 (Spray Gun) (기초)

스프레이 건은 작업자가 손에 쥐고 **도료 분사를 개폐(트리거)** 하는 장치다. 에어리스 건은 고압 도료를 다루므로 **안전장치**가 핵심이다.

### 주요 부위
| 부위 | 기능 |
|---|---|
| 트리거 | 분사 개폐 |
| 팁 가드(Tip guard) | 팁 보호 + **고압 분사 직접 접촉 방지(안전)** |
| 스위블(Swivel) | 호스 연결부 회전(작업 편의) |
| 필터(Gun filter) | 팁 직전 이물 거름(막힘 방지) |
| 트리거 락(Safety lock) | 오발사 방지 잠금 |
| 디퓨저 | 내부 유로 |

### 안전 — 고압 주입 상해(Injection injury)
- 에어리스 도료압은 **피부를 뚫고 도료를 체내로 주입**할 수 있다(수천 psi). 작은 상처처럼 보여도 **응급 외과 처치 필요**한 중대 상해.
- 규칙: 팁 청소·교체 전 **압력 해제(Pressure relief)+트리거 락**, 손을 팁 앞에 두지 않기, 팁 가드 항상 장착. 검사 입회자도 분사 경로 접근 금지.

## 2. 호스 (Hose) (심화)

에어리스 호스는 **고압 도료 전용**으로 설계된 특수 호스다.

- **구조**: 내부 튜브(도료 접촉·내용제) + 금속/섬유 보강층(Carcass) + 외부 보호 자켓.
- **안전계수(Safety factor)**: 호스는 통상 **4:1 안전계수**로 선정(최대 작업압의 4배 파열압). 평판 있는 공급자 제품 사용.
- **압력 등급**: 펌프 최대 도료압 이상의 정격 호스 사용. 손상·꺾임·노화 호스 금지(파열 시 고압 분출 위험).
- **휩 호스(Whip hose)**: 건 끝의 짧고 유연한 호스 — 손목 피로↓, 작업성↑.
- **호스 길이와 압력 강하**: 호스가 길수록 **마찰로 팁 끝 압력이 떨어진다** → 긴 호스엔 더 높은 펌프압/큰 내경 필요. 무화 불량 시 호스 길이·내경 점검.

## 3. 필터 시스템 (실무)

팁 막힘은 작업 중단·패턴 불량의 주원인 → **다단 필터**로 이물을 거른다.

| 위치 | 역할 |
|---|---|
| 흡입부(Inlet/사이펀) 스트레이너 | 굵은 이물 1차 거름 |
| 펌프 출구 고압 필터(Manifold) | 주 여과 |
| 건 필터(Gun filter) | 팁 직전 최종 거름 |

- **메시(Mesh) 선택**: 얇은 도료는 고운 메시(고번호), 두꺼운 도료는 거친 메시(저번호). 너무 고우면 고점도 도료가 막히고, 너무 거칠면 팁이 막힌다.
  - 개략: 매우 얇음 200메시 / 중간 100메시 / 프라이머 60메시 / 초고점도(방화 등) 30메시. **도료에 맞춰 선택**.
- 필터 막힘은 **압력 강하·펄세이션·테일 패턴**을 유발 → 정기 청소.

## 4. 압축공기 계통 (공압식일 때) (심화)

공압 에어리스/에어 스프레이는 **압축공기 품질**이 도막 품질·안전에 직결된다.

- **수분·유분 제거 필수**: 압축공기에 섞인 수분·오일이 도막에 들어가면 **크레이터링·핀홀·부착불량**(특히 우레탄은 수분에 민감). → **에어 드라이어/오일·워터 세퍼레이터** 설치.
- **블라스트·블로잉용 공기도 동일**: 표면 청소용 압축공기가 오염되면 표면을 오염시킨다(먼지 시험 후 블로잉 시 주의).
- 적정 공급압·유량 확보(부족하면 펌프 출력·무화 저하).

## 5. 도료 가온 장치 (Heater) (실무)

- 고형분·무용제·저온 도료는 점도가 높아 무화가 어렵다 → **도료를 가온해 점도를 낮추면** 무화·레벨링 개선, 압력 의존↓.
- 히터/가열 자켓/순환 가온 등. 단 **가온은 가사시간을 단축**(2액형) → 시간 관리 강화.

## 6. 대표 문제와 원인

| 문제 | 원인 | 대책 |
|---|---|---|
| 무화 불량(긴 호스에서) | 호스 길이·내경에 의한 압력 강하 | 펌프압↑, 큰 내경/짧은 휩 |
| 크레이터링·핀홀 | 압축공기 수분·유분 | 에어 드라이어·세퍼레이터 |
| 팁 잦은 막힘·테일 | 필터 메시 부적합·막힘 | 적정 메시·청소 |
| 트리거 작동 불량 | 건 내부 마모·이물 | 정비·필터 |
| 고압 분출/누유 | 호스 손상·노화, 연결부 풀림 | 호스 교체·체결 점검 |
| 주입 상해 | 안전수칙 미준수 | 압력해제·락·가드, 즉시 의료 |

## 8. 핵심 요약
1. 에어리스 건은 **고압 주입 상해** 위험 — 압력해제+락+팁 가드가 기본. 검사자도 분사 경로 금지.
2. 호스는 **고압 전용·4:1 안전계수**, 손상/노화 금지. **길수록 압력 강하** → 무화 불량 시 의심.
3. **다단 필터(흡입·고압·건)**, 메시는 도료 점도에 맞춤. 막힘은 펄세이션·테일의 원인.
4. (공압) **압축공기 수분·유분 제거 필수** — 크레이터링·핀홀·부착불량 예방. 표면 청소 공기도 동일.
5. 고형분/무용제는 **도료 가온**으로 무화 개선하되 가사시간 단축 주의.
