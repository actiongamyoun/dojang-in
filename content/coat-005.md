## 1. 메탈라이징/TSA란 (기초)

메탈라이징(Metallizing)은 금속을 녹여 표면에 **뿜어 붙이는 열용사(Thermal Spray)** 코팅의 통칭이다. 그중 **알루미늄을 용사**한 것이 **TSA(Thermal Sprayed Aluminium)** 다.

- 핫딥(침지)과 결정적 차이: TSA는 **녹인 금속 입자를 압축공기로 분사**해 표면에 적층한다 → 현장·대형 구조물에 적용 가능, **다공성(Porous)** 피막이 형성.
- 주 용도: **해양구조물(오프쇼어), 고온 배관, CUI(보온재하 부식) 방지, 장기 방식**. ISO 12944·NORSOK M-501 체계의 가혹환경(C5-M 등)에서 채택.
- 강점: 매우 긴 수명. 잘 관리된 TSA는 **25년 이상** 보호 가능(일반 유기 도장은 통상 5년마다 보수) → 생애주기 비용(LCC) 우위.

> 조선·해양·플랜트에서 "메탈라이징" 하면 통상 TSA(또는 아연 용사 TSZ)를 의미한다. 도장 검사원이 도장과 함께 가장 자주 만나는 용사다.

## 2. 용사 방식과 방식 원리 (심화)

### 2.1 용사 방식
| 방식 | 원리 | 특징 |
|---|---|---|
| **아크 와이어 용사(Twin-wire arc)** | 두 알루미늄 와이어를 접촉시켜 아크로 용융, 압축공기로 분사 | 증착률 높고 경제적 — 오프쇼어 표준, 품질 우수 |
| **화염 용사(Flame spray)** | 와이어/분말을 화염으로 녹여 분사 | 설비 간단 |

- 아크 용사는 **건조·청정한 압축공기**로 분사(수분·오일 혼입 금지 — 피막 품질·산화에 영향).

### 2.2 방식 메커니즘
- **희생방식(아연처럼 알루미늄이 강재보다 卑) + 차단(Barrier)** 의 복합. 특히 **고온·해수·CUI 환경**에서 알루미늄이 강재를 장기 보호.
- 아연 용사(TSZ) 대비 **내해수·고온성**이 우수 → 오프쇼어·고온 배관에 TSA 선호.

## 3. 봉공(Sealing) — TSA 위 '도장'의 본질 (실무 핵심)

TSA 피막은 **다공질**이라 그대로 두면 기공으로 부식인자가 침투할 수 있다. 그래서 **봉공(Sealing)** 을 한다 — 이것이 TSA 위 도장(실코트)의 주된 역할이다.

- **봉공재(Sealer)**: 저점도라 **기공에 침투**해야 함 → 통상 희석한 에폭시·실리콘 등 얇은 실코트. **후막 도장이 아니라 침투·봉공**이 목적.
- 원리는 무기징크 미스트 코트와 동일: 다공질 표면을 적셔 기공의 공기를 몰아내고 막는다.
- 봉공 후 필요 시 색상/추가 보호용 **톱코트**. (오프쇼어에서 'TSA + 유기 톱코트' 시스템이 가혹환경 시험에서 우수)
- 고온 용도면 **내열용 알루미늄 용사 기준(ISO 17834)** 과 내열 실러/톱코트 적합성 확인(ISO 2063-2는 -50~+200℃ 범위, 그 이상 내열은 ISO 17834 영역).

## 4. 표면처리와 검사 (실무)

### 4.1 용사 전 표면처리 — 매우 엄격
- TSA는 **기계적 결합(투묘)** 에 전적으로 의존 → 통상 **White metal(Sa 3 / NACE No.1) 청정도** + **거친 앵커 패턴(평균 조도 약 75~125㎛, 3~5 mil)** 요구. 일반 도장보다 훨씬 거친 그릿 블라스트.
- 청정도·염분·먼지 관리. 블라스트 후 **신속히 용사**(발청 전).

### 4.2 용사·봉공 검사
| 항목 | 확인 |
|---|---|
| 표면처리 | Sa 3 청정도, 조도 75~125㎛(시방), 염분/먼지 |
| 용사 피막 | **두께**(와전류/초음파/자기), **부착강도**(인장/풀오프, 시방 최소값), 외관(미용착·박리·균열), 다공도 |
| 봉공 | 침투형 실러 도포량, 봉공 완결성(기공 충전) |
| 톱코트(필요시) | 두께·외관·내열 적합성 |

- 부착 시험은 ISO 2063-2/시방의 **최소 부착강도** 기준으로 판정(풀오프, 제7강 참조). 일부 사양은 굽힘/그리드 시험도 요구.

## 5. 대표 결함과 원인

| 결함 | 원인 | 대책 |
|---|---|---|
| 용사 피막 박리 | 조도/청정도 부족, 오염, 블라스트 후 발청 | Sa 3+거친 조도, 신속 용사 |
| 기공 부식 | 봉공 미흡/누락 | 저점도 실러 충분 봉공 |
| 봉공재 들뜸 | 다공면 후막 도장(침투 불가) | 침투형 실코트 |
| 산화·품질 저하 | 습/오일 혼입 압축공기 | 건조·청정 에어 |
| 고온 박리 | 부적합 톱코트, 범위 외 온도 | 내열 사양(ISO 17834) |

## 6. 검사원 체크리스트
- [ ] 표면처리: **Sa 3(NACE 1)** 청정도 + 조도 **75~125㎛**(시방), 염분/먼지, 발청 전 용사
- [ ] 용사 방식(아크/화염), 압축공기 건조·청정
- [ ] 피막 **두께·부착강도(시방 최소값)·외관·다공도**
- [ ] 봉공: 침투형 실러, 기공 봉공 완결성
- [ ] 톱코트(필요 시) 두께·내열 적합성(ISO 17834 영역 여부)
- [ ] ISO 2063-2/NORSOK M-501/시방과 대조 기록

## 7. 핵심 요약
1. 메탈라이징/TSA = 알루미늄을 **녹여 분사**하는 열용사(핫딥 '침지'와 다름) — 오프쇼어·고온·CUI 장기 방식.
2. 방식: **희생+차단** 복합, 잘 관리 시 **25년+** 수명(LCC 우위).
3. 피막이 **다공질** → TSA 위 도장의 본질은 **봉공(Sealing)**: 저점도 실러로 기공을 막는 것.
4. 표면처리가 핵심: **Sa 3 + 조도 75~125㎛**(기계적 투묘 전적 의존), 건조·청정 압축공기.
5. 검사는 **두께·부착강도·봉공 완결성**(ISO 2063-2/NORSOK M-501). 고온은 ISO 17834 영역.
