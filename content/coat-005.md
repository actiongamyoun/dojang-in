## 1. 스프레이 팁이란 (기초)

스프레이 팁(Tip / Nozzle)은 에어리스 건 끝에서 **고압 도료를 작은 구멍(오리피스)으로 통과시켜 무화하고, 부채꼴(팬) 패턴으로 분사**하는 부품이다. 작아 보여도 **도막 두께·패턴 폭·무화 품질·도료 소모**를 좌우하는 핵심 소모품이다.

> "팁은 분무의 두뇌." 같은 펌프·같은 도료라도 팁이 틀리면 도막이 달라진다.

## 2. 팁 번호 읽는 법 — 3자리 코드 (심화)

에어리스 팁은 보통 **3자리 숫자**(예: **515**)로 표기한다.

- **첫 자리 × 2 = 팬 폭(인치)**, 표면에서 약 **12인치(30cm)** 거리 기준.
  - 예: **5**15 → 5×2 = **10인치 팬 폭**. 3xx → 6인치, 6xx → 12인치.
- **뒤 두 자리 = 오리피스(구멍) 지름**, 천분의 1인치 단위.
  - 예: 5**15** → **0.015인치** 오리피스. "17"=0.017", "21"=0.021".

> 예: **515 = 약 10인치 팬 + 0.015인치 오리피스**. (4자리 코드면 앞 두 자리가 팬 관련)

### 팬 폭과 오리피스는 독립적이다 (중요)
- **317과 517은 같은 오리피스(0.017")** 지만, **517은 10인치 팬, 317은 6인치 팬**.
- 같은 양의 도료가 더 넓은 팬(517)에 퍼지므로 **517이 더 얇은 도막**을 만든다. (317은 좁고 두껍게)
- 즉 **오리피스 = 토출량(도료량)**, **팬 폭 = 그 도료를 펼치는 너비** → 둘의 조합이 도막 두께를 결정.

## 3. 오리피스 선정 — 도료 점도에 맞춘다 (실무 핵심)

| 도료 점도 | 오리피스(개략) | 예시 |
|---|---|---|
| 얇음(스테인·래커·클리어) | 0.009~0.013" | 209~313 |
| 중간(에나멜·프라이머·일반 도료) | 0.015~0.019" | 515~519 |
| 두꺼움(고형분·고점도) | 0.021~0.027" | 521~527 |
| 초고점도(방화 PFP·퍼티·역청) | 0.031"+ | 631+ 등 |

- **얇은 도료 = 작은 오리피스**(작은 구멍으로도 무화, 너무 크면 흘러내림).
- **두꺼운 도료 = 큰 오리피스**(작은 구멍은 막힘). 단 큰 오리피스는 **큰 펌프(토출·압력)** 가 받쳐줘야 함.
- **팬 폭 = 대상 형상에 맞춤**: 넓은 면(외판·탱크 벽)은 넓은 팬(5xx~6xx), 좁은 부재(파이프·형강·에지)는 좁은 팬(2xx~4xx).
- **반드시 도료 TDS의 권장 오리피스/팬** 확인 — 본 표는 일반 가이드.

## 4. 팁 종류 (심화)

| 종류 | 특징 |
|---|---|
| **리버서블 팁(Reversible)** | 막히면 180° 돌려 역분사로 뚫고 복귀 — 현장 표준, 별도 필터 불요 |
| **파인피니시(Fine Finish)** | 미세 무화·균일 패턴(상도·외관 중시) |
| **저압/고효율(FFLP·HEA 등)** | 낮은 압력에서 무화 → 오버스프레이↓·팁/펌프 수명↑ |
| **와이드팬(Wide)** | 초광폭 팬(대면적 고속) — 큰 오리피스·고토출 필요 |
| **XHD/중코팅용** | 초고점도(텍스처·방화)용 대형 핸들 |

## 5. 팁 마모 — 도막 품질의 숨은 변수 (실무 매우 중요)

- 팁은 **텅스텐 카바이드**로 단단하지만 **소모품**이다. 연마성 도료(징크·고형분)가 고압으로 통과하며 **오리피스가 점점 커지고 팬 패턴 끝이 둥글게(짧게) 마모**된다.
- **마모된 팁의 증상**: 팬 폭이 좁아지고 가장자리가 흐려짐 → **도막이 가운데로 몰려 두꺼워지고**(불균일), **도료 소모 급증**, 오버스프레이↑.
- **결과**: WFT/DFT 불균일, 도료 낭비(비용), 외관 불량. 검사 시 **줄무늬(밴딩)·과도막/박막 혼재**가 보이면 팁 마모를 의심.
- **관리**:
  - 도장 후 **즉시 청소**(도료 굳기 전), 도료에 맞는 용제로.
  - **와이어 브러시·금속 도구로 오리피스 청소 금지** — 부드러운 솔만. 오리피스 손상은 곧 팬 불량.
  - 팬 패턴 테스트로 마모 점검, 한계 시 교체.

## 6. 팬 패턴 테스트 (실무)

- 스크랩/시험판에 분사해 **패턴이 균일한지**(가운데·가장자리 두께 차이, 꼬리(Tail), 줄무늬) 확인.
- **테일(꼬리) 패턴**(가장자리만 진하고 가운데 약함): 압력 부족 또는 팁 마모/막힘 → 압력↑ 또는 팁 점검.
- 패턴이 좌우 비대칭·갈라짐: 팁 부분 막힘 → (리버서블) 역분사 청소.

## 7. 검사원/감독 체크리스트
- [ ] 도료 TDS 권장 오리피스/팬과 장착 팁 일치
- [ ] 팬 폭이 대상 형상에 적정(대면적 넓게/좁은 부재 좁게)
- [ ] 팁 마모 상태(팬 폭 축소·가장자리 흐림) 점검
- [ ] 팬 패턴 테스트(꼬리·줄무늬·비대칭)
- [ ] 오리피스 청소 도구 적정(솔, 금속 금지)
- [ ] 마모 한계 팁 교체 이력

## 8. 핵심 요약
1. 팁 3자리 코드: **첫 자리×2=팬 폭(인치@12")**, **뒤 두 자리=오리피스(천분의 1인치)**. 예: 515=10"팬·0.015".
2. **오리피스=도료량, 팬 폭=펼침 너비** — 317과 517은 같은 구멍이나 517이 더 얇은 도막.
3. **얇은 도료=작은 오리피스, 두꺼운 도료=큰 오리피스(+큰 펌프)**. 형상에 팬 폭을 맞춘다.
4. 팁은 **카바이드 소모품** — 마모 시 팬 좁아지고 도막 불균일·도료 낭비. 줄무늬 보이면 의심.
5. **금속으로 오리피스 청소 금지(솔만)**, 즉시 청소, 팬 패턴 테스트로 관리. 권장값은 TDS 우선.

## 9. 확인 문제
1. 515 팁의 팬 폭과 오리피스 크기는?
2. 317과 517 중 더 두꺼운 도막을 만드는 것은? 이유는?
3. 마모된 팁에서 나타나는 도막 증상은?
4. 오리피스를 와이어 브러시로 청소하면 안 되는 이유는?
5. 테일(꼬리) 패턴이 나올 때의 원인과 조치는?
