> 표준: ISO 8503 시리즈(비교판/촉침), ASTM D4417(Method A 비교판 / B 깊이게이지 / C 레플리카테이프)
> ⚠️ 요구 프로파일 범위는 도료 TDS·시방 우선

---

## 1. 왜 측정하는가 (기초)

블라스트로 만든 표면의 **요철(앵커 패턴, Anchor profile)** 은 도막이 기계적으로 물고 들어가는 '투묘 깊이'다. 이 프로파일이 부착력을 좌우한다.

- **너무 얕으면**: 기계적 부착 부족 → 박리 위험.
- **너무 깊으면**: 산(peak)이 도막 밖으로 삐져나와(Peak이 DFT보다 큼) **러스트 럼블(Rust rumbling)**·조기 발청·도료 과소비.
- 그래서 시방·TDS는 **프로파일 범위(min~max)** 를 규정하고, 검사원이 이를 측정·판정한다.

> 용어 정리: '프로파일(Profile)'과 '러프니스(Roughness)'는 현장에서 거의 같은 의미로 쓰이며, 보통 **Rz(또는 Ry, peak-to-valley 높이)** 로 표현한다.

## 2. 측정 방법 3종 (심화)

![표면조도 측정](img/02_surface_profile.svg)

| 방법 | 표준 | 원리 | 특징 |
|---|---|---|---|
| **A. 비교판** | ISO 8503-1/-2, ASTM D4417 A | ISO 비교기(Grit/Shot용)와 촉·시각 비교 | 빠름·정성적, Fine/Medium/Coarse 등급 |
| **B. 깊이 게이지** | ISO 8503-5 계열, ASTM D4417 B | 첨침이 골에 닿는 깊이를 다이얼/디지털 측정 | 정량적, 휴대 간편, 현장 다용 |
| **C. 레플리카 테이프** | ASTM D4417 C | 압착 테이프에 프로파일 전사 후 두께 측정 | 정량적, 기록(증거) 보존 가능 |

> 주의: **Grit(앵귤러)용과 Shot(라운드)용 비교기는 다르다.** 사용한 연마재 형태에 맞는 비교기를 써야 한다.

## 3. 준비물

- (A) ISO 표면 비교기(연마재 형태에 맞는 것), 조명·확대경
- (B) 표면조도 깊이 게이지(디지털/다이얼), 평탄 기준판(영점용)
- (C) 레플리카 테이프(측정범위별 등급: Coarse, X-Coarse 등), 스프링식 마이크로미터, 둥근 압착 도구

## 4. 시험 절차 (상세 매뉴얼)

### 방법 B — 깊이 게이지 (현장 표준)
1. 평탄 기준판(유리/매끈면)에 게이지를 올려 **영점(Zero) 설정** — 첨침과 베이스가 같은 면일 때 0.
2. 측정면에 게이지를 수직으로 올려 첨침이 **골(valley)** 에 들어가게 한다. 베이스는 산(peak)에 얹힌다.
3. 표시값(첨침이 들어간 깊이 = 프로파일)을 읽는다.
4. **한 위치에서 여러 번**, 그리고 **여러 위치(대표면+취약부)** 에서 반복 측정 → 평균/범위 산출.

### 방법 C — 레플리카 테이프
1. 측정 범위에 맞는 **테이프 등급 선택**(예상 프로파일이 등급 범위 안에 들도록).
2. 테이프를 표면에 부착 → 둥근 도구로 **균일하게 압착**(폼이 요철에 눌려 프로파일이 전사됨, "burnishing").
3. 테이프를 떼어 **스프링식 마이크로미터**로 두께 측정.
4. **폼 기재 두께를 차감**: 폼 캐리어 자체 두께(예: 2 mil ≈ 50.8㎛)를 빼야 실제 프로파일. (장비 매뉴얼의 보정값 사용)
5. 측정 결과가 테이프 등급 범위의 끝에 걸리면 **다른 등급으로 재측정**.

### 방법 A — 비교판
1. 사용 연마재(Grit/Shot)에 맞는 비교기 선택.
2. 측정면 옆에 비교기를 대고 **촉감(긁어보기)·시각**으로 가장 비슷한 세그먼트(Fine/Medium/Coarse)를 판정.
3. 정성 등급이므로 정량 기준이 필요한 사양에는 B 또는 C로 보완.

## 5. 결과 검토 / 합부 판정

- **사양 적합성**: 도료 TDS가 요구하는 프로파일 범위(min~max) 안에 들어오는지 판정.
  - 예: 많은 중방식 하도가 약 50~85㎛급 프로파일을 권장(제품별 상이) — 정확한 값은 TDS 확인.
- **복수점 통계**: 단일 스파이크(이상치)에 휘둘리지 말 것. 평균과 범위, 최대/최소를 함께 본다.
- **과대 프로파일 주의**: 산이 도막두께(DFT)에 비해 너무 크면 peak이 노출돼 조기 발청 → DFT 설계와 함께 검토.
- 기록: 방법, 위치, 개별값/평균/범위, 사용 연마재·등급·장비, 판정.

## 6. 자주 하는 실수

| 실수 | 결과 | 예방 |
|---|---|---|
| 연마재 형태와 다른 비교기 사용 | 잘못된 등급 | Grit/Shot 맞는 비교기 |
| 레플리카 폼 두께 미차감 | 값 과대(약 50.8㎛) | 보정값 차감 |
| 등급 범위 벗어난 테이프 | 부정확 | 적정 등급 재선택 |
| 영점 미설정(게이지) | 계통 오차 | 매 측정 전 영점 |
| 단일점 측정 | 대표성 부족 | 복수점 평균/범위 |

## 8. 핵심 요약
1. 프로파일(앵커 패턴)은 부착력의 토대 — **너무 얕아도, 너무 깊어도** 문제.
2. 방법 3종: 비교판(정성)·깊이게이지(정량, 현장 표준)·레플리카테이프(정량+기록).
3. **연마재 형태에 맞는 비교기**, **레플리카 폼 두께 차감**이 정확도 핵심.
4. 단일점 아닌 복수점 통계로 판정, DFT 설계와 함께 본다.
5. 요구 범위는 TDS/시방 우선.
