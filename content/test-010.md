## 1. 왜 먼지를 측정하는가 (기초)

블라스트(연마분사) 후 강재 표면에는 연마재 잔사, 분진, 녹가루 같은 **느슨한 먼지(Loose dust)** 가 남는다. 이 먼지는 도막과 강재 사이에 끼어:

- **부착력을 떨어뜨리고**(먼지 위에 도료가 얹히는 꼴),
- **수분을 흡착·보유**해 도막하 부식을 촉진한다.

그래서 도장 사양서는 통상 "모든 표면은 오일·그리스·먼지·수용성염이 없어야 한다"고 규정하며, **도장 직전에 먼지도를 측정**해 관리한다.

> 핵심: 먼지는 **수평면, 파이프 내부, 구조물 공동부(캐비티)** 에 더 잘 쌓인다. 이런 부위는 특별히 신경 써서 검사해야 한다.

## 2. 측정 원리 (심화)

ISO 8502-3은 **감압(점착) 테이프를 표면에 눌러 붙였다 떼어, 테이프에 묻은 먼지를 표준 사진과 비교**하는 정성(Qualitative) 시험이다.

평가는 **두 가지**를 본다:
1. **먼지 양(Quantity rating)**: 0~5 등급 — 표준 그림(Pictorial)과 비교. (숫자가 클수록 먼지 많음)
2. **먼지 크기(Particle size class)**: 1~5 클래스 — 입자 크기를 서술적 분류.

| 크기 클래스 | 설명(개략) |
|---|---|
| 1 | 정상 시력으로 보이지 않는 입자(확대해야 보임) |
| 2 | 정상 시력으로 겨우 보이는 입자 |
| 3 | 정상 시력으로 분명히 보이는 입자(약 0.5mm 이하) |
| 4 | 약 0.5~2.5mm 입자 |
| 5 | 약 2.5mm 초과 입자 |

> 양(0~5)과 크기(1~5)는 별개 척도다. 보고는 보통 "**양 등급 / 크기 클래스**" 형태로 쓴다(예: "2 / 2"). 중간 반단계(예: 2.5)를 쓸 수도 있다.

## 3. 준비물

- ISO 8502-3 규정 점착테이프(폭·점착력이 규정된 것)
- 대조 배경판: 먼지가 잘 보이도록 — **밝은 먼지엔 어두운 판, 어두운 먼지엔 밝은 판**
- 표준 비교 사진(등급 도해), 롤러 또는 손가락 압착, 기록지(테이프 보존용)

## 4. 시험 절차 (상세 매뉴얼)

1. **위치 선정**: 대표면 + **먼지가 쌓이기 쉬운 곳**(수평면, 하부, 구석, 파이프 내부). 1점이 아닌 복수점.
2. **테이프 부착**: 규정 길이의 테이프를 표면에 붙이고, **엄지로 일정 압력으로 문질러** 밀착(규정된 방식대로 일정 길이를 따라 압착).
3. **테이프 제거**: 테이프를 떼어낸다(먼지가 점착면에 옮겨 붙음).
4. **대조판에 부착**: 떼어낸 테이프를 **대조 색 배경판**에 붙인다.
5. **양 등급 판정**: 표준 그림과 비교해 먼지 양을 0~5로 판정.
6. **크기 클래스 판정**: 입자 크기를 1~5로 판정(필요 시 확대경).
7. **기록·보존**: 등급/클래스, 위치, 사용 테이프를 기록하고 **테이프 자체를 증빙으로 보존** 가능.

## 5. 결과 검토 / 합부 판정

- **합격 기준은 시방 우선.** 실무 통념상 **양 등급 2 이하를 합격**으로 보는 경우가 많고, **3·4·5는 청소 후 재시험** 대상이다. (크기 클래스는 사양에서 요구하지 않는 경우가 흔하다.)
- 불합격 시: **압축공기 블로잉(통상 90~120 psi, 수분·유분 없는 청정 공기)** 또는 진공 청소 후 재측정. 압축공기 자체에 수분·오일이 섞이면 오히려 오염되므로 에어 청정도 확인.
- 기록: 위치, 양 등급/크기 클래스, 대조판 색, 판정, (가능 시) 테이프 보존.

> 주의: 이 시험은 **정성·비교 시험**이라 판정자 간 편차가 있을 수 있다. **표준 사진을 곁에 두고** 일관된 기준으로 보는 것이 신뢰성의 핵심이다.

## 6. 자주 하는 실수

| 실수 | 결과 | 예방 |
|---|---|---|
| 대조판 색을 안 맞춤 | 먼지 안 보여 과소평가 | 먼지색 반대 배경판 |
| 수평면·구석 누락 | 먼지 많은 곳 놓침 | 쌓이기 쉬운 부위 포함 |
| 압착 불균일 | 먼지 전사 부족→과소 | 일정 압력·규정 방식 |
| 오염된 압축공기로 청소 | 오일·수분 재오염 | 에어 청정도 확인 |
| 양/크기 척도 혼동 | 등급 오기재 | "양/크기" 분리 표기 |

## 7. 검사원 체크리스트
- [ ] 도장 직전 측정, 복수 위치(수평·구석·파이프 내부 포함)
- [ ] 규정 테이프·일정 압착, 먼지색 반대 대조판
- [ ] 양 등급(0~5)·크기 클래스(1~5) 분리 판정
- [ ] 표준 사진과 비교, 시방 합격선과 대조
- [ ] 불합격 시 청정 압축공기/진공 청소 후 재측정
- [ ] 등급·위치 기록, 테이프 증빙 보존

## 8. 핵심 요약
1. 먼지는 부착 저하·수분 보유로 도막 실패를 부른다 → **도장 직전 측정**.
2. ISO 8502-3 = 점착테이프로 먼지를 떼어 **표준 사진과 비교**하는 정성 시험.
3. **양(0~5)** 과 **크기(1~5)** 를 분리 평가, "2/2"처럼 표기.
4. 통상 **양 2 이하 합격**, 3 이상은 청소 후 재시험(기준은 시방 우선).
5. 대조판 색 맞추기 + 표준 사진 병행이 신뢰성의 핵심. 수평·구석·파이프 내부를 놓치지 말 것.

## 9. 확인 문제
1. 먼지가 도막 실패를 일으키는 두 메커니즘은? (부착 저하 + 수분 보유)
2. ISO 8502-3에서 평가하는 두 가지 항목과 각 척도는? (양 0~5 / 크기 1~5)
3. 통상적인 합격 등급과, 불합격 시 조치는?
4. 대조 배경판의 색을 먼지색과 반대로 하는 이유는?
