## 1. FROSIO 개요

**FROSIO**(Faglig Råd for Opplæring og Sertifisering av Inspektører innen Overflatebehandling)는 노르웨이의 표면처리 검사원 교육·인증 협의회로, 1986년 설립되었다. 북해 해양플랜트의 부식 문제 대응에서 출발해 현재는 NACE(AMPP)와 함께 **도장 검사 분야 세계 양대 자격**으로 인정받는다.

- **NORSOK M-501** 표준에서 검사원 자격 요건으로 FROSIO Level III(또는 NACE Level 3)를 명시 → 유럽/북해/조선해양 프로젝트의 사실상 표준
- 조선소 기준: 유럽 선주·오프쇼어 물량에서 특히 선호도 높음

---

## 2. 자격 등급 체계

**시험은 하나, 등급은 경력으로 결정**되는 구조 (NACE와의 가장 큰 차이점)

| 등급 | 색상 | 요건 | 권한 |
|---|---|---|---|
| **Level I** | White | 시험 합격 + 관련 경력 없음/부족 | 단독 검사 불가, 상위 검사원 감독 하 업무 |
| **Level II** | Green | 시험 합격 + 관련 경력 최소 2년 (검사 경력 2년 미만) | 검사 업무 수행 가능 |
| **Level III** | Red | 시험 합격 + 관련 경력 최소 5년 (그중 검사 경력 최소 2년 증빙) | 단독 검사원 완전 권한, NORSOK 요구 등급 |

- Level I, II는 **5년 이내에 상위 등급으로 업그레이드** 필요
- 경력 증빙: 이력서(CV) + 고용주 확인서 + 레퍼런스 리스트
- 인정 경력 범위: 도장 계획·생산 / 도장 시공 / 도장 검사 관련 직무
- **경력 5년 이상 보유자는 합격 즉시 Level III(레드) 발급 가능**

## 3. 인증 요건 (4가지 모두 충족)

1. **Training**: FROSIO 공인 교육기관의 80시간 교육 과정 이수
2. **Knowledge**: 이론 시험 합격
3. **Skills**: 실기 시험 합격
4. **Experience**: 관련 경력 서류 증빙 (등급 결정용)

---

## 4. 시험 구성 및 합격 기준 (2026 최신)

### 시험 구조
- **이론(Theory)**: 4시간 / **실기(Practical)**: 4시간
- 도장검사원(Surface Treatment) 이론 문항 수: 약 80~90문항
- 실기: 10개 테이블에 각각 실기 과제 배치, 순환하며 수행
- 시험 언어: 영어 (한국 개설 과정 기준)

### 합격 기준

| 구분 | 만점 | 합격선 | 비고 |
|---|---|---|---|
| 이론 | 150점 | **100점 (66.6%)** | 95~99점은 제2채점자 재평가 |
| 실기 | 48점 | **32점 (66.6%)** | 29~31점은 재평가 |

- 두 시험 **모두 합격**해야 함
- 채점은 독립 채점관이 수행, FROSIO 감독관 입회 하 시험 진행 (강사 포함 누구도 시험장 출입 불가)
- 결과는 시험 후 **약 3~4주** 뒤 등록 이메일로 개별 통보
- 채점 결과에 대한 서면 이의신청 가능 (FROSIO 사무국)

### 시험장 규정
- **지참 가능**: 연필/펜, 자, 비프로그래밍 계산기, 종이 사전(감독관 검사)
- **금지**: 휴대폰, 스마트워치, AI 글래스(Ray-Ban Meta 등), 전자사전, 부정행위 목적 메모
- 위반 시 실격 + **향후 FROSIO 시험 응시 불가** 가능
- 2023년부터 종이 수험표 폐지 → **디지털 등록제** (한글명, 영문명, 이메일 — 네이버 메일 불가 — 증명사진 등 제출)

### 재시험(Re-exam)
- 불합격 파트(이론 또는 실기)만 재응시
- 재시험 응시료는 이론/실기/둘 다 동일
- **최초 시험 후 3년 이내** 언제든 재응시 가능
- 타 교육기관에서 불합격한 경우에도 재시험 신청 가능

## 5. 교육 과정 내용 (80시간)

| 영역 | 주요 내용 |
|---|---|
| 부식 이론 | 부식 메커니즘, 전기화학, 음극방식 |
| 표면 전처리 | 블라스팅, 연마재, Sa/St 등급, ISO 8501/8502/8503 |
| 도료 화학 | 에폭시, PU, 징크 등 도료 유형별 특성, 경화 메커니즘 |
| 도장 시공 | 에어리스 스프레이, 도장 조건 관리 |
| 검사 실무 | DFT 측정, 노점 계산, 염분 측정, 핀홀 테스트, 접착력 시험 |
| 표준/시방 | ISO 12944, ISO 19840, NORSOK M-501 등 |
| 보고서 | 검사 보고서 작성, 검사원 역할과 윤리 |

> 참고: FROSIO 공식 교재(FROSIO Book)는 frosiobook.no 에서 **무료 다운로드** 가능 — 시험 대비 및 일반 참고용

## 7. 자격 유지 및 갱신

- 유효기간: **최초 발급일로부터 5년**
- 갱신 책임은 검사원 본인 (FROSIO 사무국에 기한 내 통보)
- 갱신은 frosio.no 개인 포털 페이지에서 직접 진행
- **Level III 갱신 요건**: 5년 내 최소 2년의 검사 경력 증빙
  - 최신 CV + 외부 연락처 포함 레퍼런스 리스트 + 고용주 확인서
- 갱신 수수료 별도
- 모든 자격증은 **디지털 발급** — frosio.no에서 누구나 VERIFY 버튼으로 진위 확인 가능
- 자격 만료 후에는 이론+실기 시험 재응시 필요

## 8. NACE(AMPP CIP)와 비교

| 구분 | FROSIO | AMPP(NACE) CIP |
|---|---|---|
| 발급국 | 노르웨이 | 미국 |
| 시험 구조 | 단일 시험 + 경력으로 등급 결정 | Level 1, 2 각각 별도 과정·시험 |
| 등급 | I(White) / II(Green) / III(Red) | Level 1 / 2 / 3 |
| 강세 분야 | 유럽, 북해, 조선·해양(NORSOK) | 미주, 중동, 육상 플랜트 |
| 교육 | 80시간 (한국: 온라인 1개월 + 실습) | 레벨당 약 1주 |
| 유효기간 | 5년 | 3년 (포인트 갱신제) |
| 한국 비용 | 교육 국비 지원 + 응시료 개인 부담 | 전액 개인/회사 부담 (수백만 원대) |

## 9. 응시 전략 요약 (현직 도장 검사 경력자 기준)

1. **경력 5년+ 검사 경력 2년+ → 합격 즉시 Level III** 발급 가능
2. 평소 실무(노점 계산, DFT, 블라스팅 등급 판정, ISO 8501/8502)가 시험 범위와 직결 → 실기 유리
3. 이론은 영어 시험 — 도료 화학/부식 이론의 **영문 기술용어** 보강이 관건
4. FROSIO Book 무료 교재로 사전 학습 가능
5. 가장 가까운 일정: **2026년 7월 Class A/B** (신청 마감 임박) 또는 **9월 Class C**
6. 신청·문의: jh.park@kitr.co.kr
