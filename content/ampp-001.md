## 1. AMPP CIP 개요

**CIP**(Coating Inspector Program)는 미국 **AMPP**(Association for Materials Protection and Performance)가 운영하는 도장검사원 자격 프로그램으로, FROSIO와 함께 세계 양대 도장 검사 자격이다.

### ★ 중요: NACE → AMPP 통합 (반드시 알아야 할 변경사항)
- 2021년 **NACE International**(부식기술자협회)과 **SSPC**(철구조물도장협의회)가 합병하여 **AMPP** 출범
- 기존 **NACE CIP**와 **SSPC PCI** 프로그램이 **AMPP CIP**로 통합됨
- 본부: 미국 휴스턴/피츠버그, 130개국 이상 활동, 회원 4만 명+
- **기존 NACE CIP / SSPC PCI 자격은 업계가 AMPP 자격을 완전히 채택할 때까지 계속 유효** — 갱신도 가능
- 신규 응시자는 AMPP 명칭의 자격을 취득하게 됨

### 자격 명칭 변경 (구 ↔ 신 대응표)

| 구 명칭 | 신 AMPP 명칭 |
|---|---|
| NACE CIP Level 1 / SSPC PCI Level 1 | **Basic Coatings Inspector** (CIP Level 1) |
| NACE CIP Level 2 / SSPC PCI Level 2 | **Certified Coatings Inspector** (CIP Level 2) |
| NACE CIP Level 3 / SSPC PCI Level 3 | **Senior Certified Coatings Inspector** (CIP Level 3) |

---

## 2. 자격 등급 체계

**FROSIO와의 핵심 차이: 레벨마다 별도의 교육 과정 + 별도의 시험**을 순차적으로 통과해야 한다.

| 등급 | 자격명 | 역할/권한 |
|---|---|---|
| **Level 1** | Basic Coatings Inspector | 비파괴 기법·계측기를 사용한 기본 도장 검사 수행 |
| **Level 2** | Certified Coatings Inspector | 비파괴+파괴 검사 수행·문서화 (Shop 단독 / Field는 L3 감독 하) |
| **Level 3** | Senior Certified Coatings Inspector | 현장 리더. L1·L2 검사원 감독, 현장 문제 해결, 표준 적용 판단 |

---

## 3. 레벨별 인증 요건 (2026년 3월판 EPG 기준)

### Level 1 — Basic Coatings Inspector
1. **CIP Level 1 코스** 이수
2. **윤리 교육**: AMPP "Ethics for the Corrosion Professional" 코스 이수 (또는 동등 과정 승인)
3. **Professional Code of Conduct** 서약 (My Certification Portal)
4. **실기 시험**: CIP Level 1 Practical (코스 마지막 날 현장 시행)
5. **이론 시험**: Basic Coatings Inspector CBT (Pearson VUE 컴퓨터 시험, 코스와 별도 일정)
6. 인증 신청서 제출
- **갱신: 3년마다** 재인증 신청 필요 (도장 검사 분야 1.5년 실무 경력 요건)

### Level 2 — Certified Coatings Inspector
- CIP Level 2 코스 이수 + 실기/이론 시험 합격
- 고급 부식 이론(음극방식 병용), 환경 측정 장비(디지털 습도계, 데이터로거), 원심 블라스트·워터젯 검사 등 심화 내용
- 케이스 스터디 기반 문제 해결·팀 역학 훈련 포함

### Level 3 — Senior Certified Coatings Inspector
- **전제 조건: Certified Coatings Inspector(L2) 자격 보유**
- 윤리 코스 이수
- **구술 시험(Oral Exam)**: 3인 심사위원 패널 앞에서 진행
  - CIP 1·2 Body of Knowledge 기반
  - 기술 문항 6개 + 실무 문항 4개에 구두 답변
  - 섹션당 1시간 배정
  - 현장 트러블슈팅, L1/L2 감독 능력, 표준 근거 제시 능력 평가
- 평가 도메인에 그룹 행동 관리, 분쟁 상황 중 기술적 제안, 윤리적 압박 상황 대응 등 **리더십·윤리 영역 포함**

---

## 4. 시험 방식 (2026 최신)

| 구분 | 방식 |
|---|---|
| 실기 시험 | 코스 마지막 날 교육장에서 시행 (hands-on) |
| 이론 시험 | **CBT (Pearson VUE)** — 코스와 별도로 시험센터에서 응시 |
| 최초 CBT 응시권 | 코스 등록비에 포함 |
| CBT 유효기간 | 발급 후 1년 (또는 코스 종료일 기준) — 기간 내 1회 응시 |
| 재시험 | 별도 비용 발생, 실기 재시험은 코스 재참석이 요구될 수 있음 |
| 계산기 | CBT 화면 내 TI 계산기 제공 (요청 시 실물 공학용 계산기 제공) |
| Level 3 | 구술 시험 (3인 패널) |

- 결제: AMPP 웹사이트 또는 My Certification Portal (카드/수표/PayPal/계좌이체)

## 5. 교육 과정 구성

### CIP Level 1 (2022년 전면 개편판)
- NACE CIP + SSPC PCI의 장점을 통합해 재설계, 실습 비중 대폭 강화
- 내용: 도료 재료, 표면 전처리, 도장 기법, 검사 시험(비파괴), 계측기 사용
- **Blended(혼합) 형식 제공**: Part I 가상 강의실 2일(실시간 강사) + Part II 대면 실습 3일(도장 랩, 시연)
- 전통적 대면 형식도 운영 (통상 5~6일)
- 연간 전 세계 5,000명 이상 수강

### CIP Level 2
- 강의 + 토론 + 그룹 실습 + 핸즈온 랩 + 실제 사례 기반 케이스 스터디
- 파괴 검사 포함, 특수 환경(원심 블라스트, 워터젯) 검사

## 6. 한국 응시

- AMPP는 한국에서도 정기적으로 CIP 코스를 개설해 왔다 (서울/울산/거제 등 조선·플랜트 밀집 지역 위주, AMPP Korea 지역 조직 경유)
- **2026년 한국 개설 일정은 이번 검색에서 확정 정보를 찾지 못함** → ampp.org 코스 파인더에서 "South Korea" 필터로 직접 확인 필요:
  - https://www.ampp.org/education/education-resources/courses-by-program/coating-inspector-program
- 이론 CBT는 Pearson VUE 시험센터에서 응시 (한국 내 다수 센터 운영)
- 비용: 레벨당 교육+시험 포함 **대략 $2,500~3,500 수준**으로 알려져 있으나, 정확한 금액은 ampp.org 코스별 페이지에서 확인 필요 (회원/비회원 가격 상이)
- FROSIO(KRISO 국비 과정)와 달리 **전액 개인/회사 부담**

---

## 7. 자격 유지 및 갱신

- **3년마다 재인증(Recertification) 신청** 필요
- Level 1 기준: 도장 검사 분야 1.5년 실무 경력 증빙
- 갱신은 My Certification Portal에서 진행
- 기존 NACE CIP / SSPC PCI 보유자: 전환기 동안 기존 자격 유효, 갱신 시점에 AMPP 체계로 전환 안내

## 8. FROSIO와 비교

| 구분 | AMPP CIP | FROSIO |
|---|---|---|
| 발급 | 미국 AMPP (NACE+SSPC 통합) | 노르웨이 FROSIO |
| 시험 구조 | **레벨별 별도 코스+시험** (L1→L2→L3 순차) | **단일 시험 + 경력으로 등급 결정** |
| L3 도달 | L1, L2 모두 거쳐야 함 (구술 시험) | 경력 5년+ 이면 1회 합격으로 즉시 LV III |
| 이론 시험 | CBT (Pearson VUE, 별도 일정) | 지필 4시간 (교육 직후) |
| 갱신 주기 | 3년 | 5년 |
| 강세 분야 | 미주, 중동, 육상 플랜트, 파이프라인 | 유럽, 북해, 조선·해양 (NORSOK) |
| 한국 비용 | 레벨당 수백만 원 (전액 자부담) | 교육 국비 + 응시료만 자부담 (KRISO) |
| 윤리 요건 | 윤리 코스 필수 + Code of Conduct 서약 | 별도 윤리 코스 없음 |

### 경력자 관점 핵심 포인트
- **검사 경력 5년+ 보유자**: FROSIO는 시험 1회로 최고 등급(LV III) 도달 가능 → **시간·비용 효율 압도적**
- AMPP CIP는 L3까지 코스 3개 + 시험을 순차 통과해야 하므로 기간과 비용이 몇 배
- 단, 발주처가 "NACE/AMPP CIP Level 2 이상"을 명시하는 프로젝트(미국계, 중동 플랜트)라면 CIP가 필수
- 조선소 유럽 선주 물량 중심이면 FROSIO 우선, 커리어 확장(플랜트·해외) 고려 시 둘 다 보유가 이상적

---

## 9. 응시 전략 요약 (현직 도장 검사 경력자 기준)

1. 우선순위 판단: 소속/목표 발주처 시방서가 어느 자격을 명시하는지 확인
2. AMPP로 가는 경우 L1부터 시작 — 실무 경력으로 L1 면제는 불가 (코스 이수 필수)
3. 이론 CBT는 코스와 별도 일정 → 코스 직후 1년 내 응시 (응시권 만료 주의)
4. 영어 시험 — 도료/부식 영문 기술용어 학습 필요 (FROSIO와 공통)
5. 윤리 코스(Ethics for the Corrosion Professional)는 온라인 이수 가능 — 미리 처리 가능
6. 한국 일정·정확한 비용은 ampp.org 코스 파인더 + AMPP Korea 문의로 확정
