## 1. 도장 전 장비 세팅 흐름 (기초)

도장 품질의 상당 부분이 **분사 전 세팅**에서 결정된다. 표준 순서:

1. **도료 준비**: 혼합비·숙성(2액)·점도(필요 시 가온/지정 신너), 거름(Strainer).
2. **장비 선정·점검**: 펌프(압력비/토출), 팁(오리피스/팬), 호스(정격·길이), 필터(메시), (공압) 에어 드라이어.
3. **압력 설정**: 무화에 필요한 **최저 압력**부터 시작 → 패턴 보며 조정.
4. **팬 패턴 테스트**: 스크랩에 분사해 패턴 균일·꼬리·줄무늬 확인.
5. **환경 확인**(검사 연계): 온도·습도·노점(ΔT≥3℃), 표면 청정·조도.

## 2. 올바른 분사 기법 (심화) — 도막 균일의 기본

| 항목 | 원칙 |
|---|---|
| **거리** | 표면에서 일정 거리(통상 약 30cm/12") 유지 — 가까우면 두껍고 튀고, 멀면 드라이 스프레이 |
| **각도** | 표면에 **수직**으로 분사. 손목을 꺾어 부채질(Arcing)하면 양끝이 얇아짐 |
| **속도** | 일정한 손 이동 속도 — 빠르면 박막, 느리면 과막/새깅 |
| **중첩(Overlap)** | 패스마다 **약 50% 중첩** → 줄무늬·얼룩 방지 |
| **트리거링** | 패스 시작 전 트리거, 끝나고 놓기(가장자리 두께 보정) |
| **스트라이프 코트** | 에지·용접부·구석 선행 도장(취약부 두께 확보) |

> 검사원이 시공을 볼 때 **거리·수직·속도·중첩·스트라이프**를 확인하면 도막 불균일을 사전에 잡는다.

## 3. WFT 관리와 장비의 관계 (실무)

- 목표 **DFT = WFT × 부피고형분(VS%)**. 시공자는 **WFT 게이지**로 실시간 두께를 확인하며 분사.
- 팁 오리피스(도료량)·팬 폭·손 속도·중첩이 WFT를 좌우 → **장비 세팅이 곧 두께 관리**.
- 검사원은 WFT 측정 습관과 최종 DFT(ISO 19840/SSPC-PA2)로 교차 확인.

## 4. 분사 결함 트러블슈팅 (실무 핵심)

### 4.1 패턴/무화 결함
| 증상 | 원인 | 조치 |
|---|---|---|
| **테일(꼬리) 패턴** | 압력 부족, 팁 마모/막힘, 고점도 | 압력↑(한계 내), 팁 청소/교체, 가온/희석 |
| **줄무늬(밴딩)** | 중첩 부족, 팬 마모, 거리/속도 불균일 | 50% 중첩, 팁 교체, 기법 교정 |
| **굵은 입자(무화 불량)** | 압력 부족, 팁 큼, 점도 높음 | 압력·팁·점도 조정 |
| **패턴 비대칭/갈라짐** | 팁 부분 막힘 | (리버서블) 역분사 청소 |

### 4.2 도막 결함과 장비 원인
| 도막 결함 | 장비/세팅 원인 | 조치 |
|---|---|---|
| 드라이 스프레이 | 분사 거리 멀다, 압력↑과다(미세건조), 신너 과휘발 | 거리·압력·신너 조정 |
| 새깅(흘러내림) | 과도막(느린 속도·큰 팁), 과압 근접 | 속도·팁·거리 |
| 오렌지필 | 무화 부족·점도/온도 | 압력·점도·가온 |
| 핀홀/크레이터 | 압축공기 수분·유분, 표면 오염 | 에어 드라이어, 표면 청정 |
| 펄세이션 줄무늬 | 펌프 압력 출렁임(흡입공기·필터 막힘) | 흡입부·필터·정비 |

> 진단 원칙: **결함 형태 → 가능한 장비/세팅 원인 역추적 → 한 번에 하나씩 조정**(압력→팁→점도→기법 순으로). 시험판에서 확인 후 본 도장.

## 5. 소모품(Wear parts) 관리 (실무)

| 소모품 | 마모/수명 신호 | 관리 |
|---|---|---|
| **팁** | 팬 폭 축소·가장자리 흐림·도료 소모↑ | 즉시 청소(솔, 금속 금지), 패턴 테스트, 한계 시 교체 |
| **펌프 패킹(V-packing)** | 누유·압력 유지 불가 | 리페어 키트로 교체 |
| **볼/시트(Check valve)** | 흡입/토출 불량·펄세이션 | 정비·교체 |
| **필터(메시)** | 압력 강하·테일·막힘 | 도료별 메시·정기 청소/교체 |
| **호스** | 손상·꺾임·노화·누유 | 정격 확인·교체(안전) |
| **건 패킹/니들** | 트리거 누유·작동 불량 | 정비 |

## 6. 세척과 보관 (실무)

- **즉시 세척**: 작업 후(특히 2액 가사시간 내) 도료에 맞는 용제로 전 계통 플러싱 — 굳으면 펌프·팁·필터 손상.
- **2액 도료**: 가사시간 경과분은 폐기, 굳기 전 세척 필수(굳으면 막힘·정비 비용).
- **팁/필터 보관**: 용제/펌프 아머에 담가 건조·손상 방지.
- **장비 보관**: 펌프 내 보호액(Pump armor) 충전(동절기 동결·부식 방지).

## 7. 안전 종합 (필수)

- **고압 주입 상해**: 압력해제+트리거 락+팁 가드. 손을 분사 경로에 두지 않기. 검사자도 접근 금지.
- **밀폐구역(탱크) 도장**: 용제 증기·산소결핍·폭발하한(LEL) — 환기·가스측정·송기마스크. 검사 입회 시 동일 보호.
- **이소시아네이트(우레탄)/특수 도료**: 호흡보호. **압축공기·전기 방폭** 관리.

## 8. 검사원/감독 종합 체크리스트
- [ ] 도료 준비(혼합비·숙성·점도·거름)
- [ ] 장비 정합(펌프 압력비/토출·팁 오리피스/팬·호스 정격·필터 메시)
- [ ] 최저 무화 압력 세팅·팬 패턴 테스트
- [ ] 분사 기법(거리·수직·속도·50% 중첩·스트라이프)
- [ ] WFT 실시간 관리 → DFT 교차 확인
- [ ] 결함 발생 시 원인 역추적(압력→팁→점도→기법)
- [ ] 소모품 마모/교체 이력, 작업 후 즉시 세척
- [ ] 안전(주입상해·밀폐구역·방폭·호흡보호)

## 9. 핵심 요약
1. 품질은 **세팅에서 시작**: 도료준비→장비정합→최저압력→패턴테스트→환경확인.
2. 분사 기법 5원칙: **일정 거리·수직·일정 속도·50% 중첩·스트라이프 코트**.
3. **DFT=WFT×VS%** — 장비 세팅(팁·팬·속도·중첩)이 곧 두께 관리. WFT 게이지로 실시간 확인.
4. 결함은 **형태→장비 원인 역추적**: 테일=압력/팁, 줄무늬=중첩/팬, 핀홀=공기 수분, 펄세이션=펌프/필터.
5. **소모품 관리(팁·패킹·필터·호스)+즉시 세척+안전(주입상해·밀폐구역)** 이 장비 운용의 마무리.
